# Logos de Equipos

Coloca aquí los logos de los equipos de La Liga.

## Formato de nombre
Los archivos deben nombrarse según el ID del equipo que devuelve la API:
- `{id_equipo}.png`

## Ejemplos
- `529.png` - FC Barcelona
- `541.png` - Real Madrid
- `530.png` - Atlético Madrid
- etc.

## Formato recomendado
- PNG con fondo transparente
- Tamaño: 64x64px o 128x128px
- Cuadrado para mejor visualización
